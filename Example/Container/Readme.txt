1) Make sure you have docker and docker compose installed :)
2) Navigate on a terminal to location where you downloaded this folder
3) If in linux> sudo docker-compose up --build  if in windows> docker-compose up --build
4) Wait till it's done configuring and installing dependencies.
5) Once its completed the install and no error were found you are good to start loading data to the db
6) Check the readme in the documentation folder in the main branch for details about how to load data to the database. Especially the session
 "Processing Test Suites and Loading to the Database (Using pargo for implemented library test suite parsers; rocFFT, rocBLAS, rocRAND)"
7) docker-compose ps < this is a comand to check if containers are alive and port specifications.
